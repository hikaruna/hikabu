# README

自作の株管理ツール
スプレッドシートで無理やり管理しているものをRDBに落とすのが目的.
システムの本体をDBに寄せたい.
railsはDBのスキームの変更をバージョン管理するための手頃なRDBマネージャーとして導入しているのみ.
迷っているが、最終的にsqlite-wasmかなにかでブラウザアプリとする可能性を見て、DBはsqliteで行こうと思う. (機能的にpostgresqlも捨てがたくて迷った)
